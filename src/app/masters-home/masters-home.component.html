List of Schools Masters

This feature provides the user a huge list of schools to select from. The schools are sorted based on their overall global ranking. User are provided with a list of schools, a small description about them and a search bar to narrow down their choices. Moreover, if users are interested in a school, they can add schools as dream, reach or safe option.
